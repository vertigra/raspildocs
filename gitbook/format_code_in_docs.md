# Соглашение по форматированию кода Markdown в документах

---

## Заголовок статьи

```markdown

# Название статьи
(пустая строка)
---
(пустая строка)

``

На конце не должно быть точки.

## Подзаголовки в статье

```markdown

(пустая строка)
## Подзаголовок
(пустая строка)

``

На конце не должно быть точки.

## Коментарии/todo

```markdown

(пустая строка)
<!-- todo сделать что-нибудь -->
(пустая строка)

``

## Оформление блоков кода

```java

If(int i = 0; i < 10; i++ ){
    do somphing
}
