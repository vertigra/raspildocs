# Соглашение по форматированию кода Markdown в документах

---

## Заголовок статьи

```md

# Название статьи
(пустая строка)
---
(пустая строка)

```

На конце заголовка не должно быть точки.

## Подзаголовки в статье

```md

(пустая строка)
## Подзаголовок
(пустая строка)

```

На конце заголовка не должно быть точки.

## Коментарии/todo

```md

(пустая строка)
<!-- todo сделать что-нибудь -->
(пустая строка)

```

## Оформление блоков кода

```md

/```java

If(int i = 0; i < 10; i++ ){
    do somphing
}

/```

```
